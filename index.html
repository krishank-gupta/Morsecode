Morse Code broke the barriers of communication over long distances

In the early 1800s, many scientists and engineers were trying to find a way to create a device that could be used to communicate over large distances. In 1836, Samuel Morse, Joseph Henry & Alfred Vail invented an electrical telegraph system. It was the first of its kind. However, this telegraph system could only send electrical pulses to another machine. Therefore, humans wouldn't be able to understand the message through pulses, so a code needed to be developed that could translate these pulses into alphabets. This was developed by Samuel Morse.


Image Credits: Wikipedia

The code utilizes standarized sequences of two diffrent pulse lengths called dits and dahs. The dits are represented by a • or sometimes a . and the dahs are represented by -. Therefore, the letter A can be represented as •- or dit dah.

This is a telegraph. This machine creates or receives the electrical pulses. These pulses are then translated by the operator. To produce a dit, you press the black button for 1 unit and to produce a dah, you press th black button for 3 units. The dot duration in one unit and the basic unit of time measurement in Morse Code.

The Birth of Morse Code

In the late 1940s, the original Morse Code was starting to be used widely all around the world. Since it was made for the transmission of English alphabets, it was clear that it was inadequate for the transmission of non-English text, since it lacked codes for alphabets with accents. To overcome this defieceincy, the International Morse Code was born. The International Morse Code was the result of a conference of European Nations in 1851. This new code was suitable for languages like French. This new code is also called the Continental Morse Code.

The Baltimore-Washington Telegraph Line

The Baltimore-Washington Telegraph Line was the first long-distance telegraph line set up to communicate between Washington and Baltimore. In March 1843, the US Congress allocated 30,000 USD to set up a telegraph line between Washtington and Baltimore. Morse tried to lay the wires underground but it failed after 15 kilometres. He finally decided to set up poles just like they did in England. This new line was demonstrated on May 24, 1844 from the United States Capital in Washington to the Mount Clare Station of the railroad in Baltimore.

What Hath God Wraught

"In a demonstration witnessed by members of Congress, American inventor Samuel F.B. Morse dispatches a telegraph message from the U.S. Capitol to Alfred Vail at a railroad station in Baltimore, Maryland. The message–“What Hath God Wrought?”–was telegraphed back to the Capitol a moment later by Vail. The question, taken from the Bible (Numbers 23:23), had been suggested to Morse by Annie Ellworth, the daughter of the commissioner of patents."
